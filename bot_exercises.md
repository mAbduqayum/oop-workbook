# Python Telegram Bot Exercises

## Easy
- [ ] Echo Bot - Create a bot that repeats back any message the user sends. This teaches basic message handling and responding.
- [ ] Welcome Bot - Build a bot that sends a welcome message when a user starts the bot with `/start` and shows help information with `/help` command.
- [ ] Inline Calculator

## Medium
- [ ] PEMDAS Calculator - Create a bot that accepts simple math expression and returns the result. Example: /calc 5 + 3. Note: using eval is insecure
- [ ] Reminder Bot - Make a bot that allows users to set reminders. The bot should message them after a specified time period.

## Hard
- [ ] Image Processing Bot - Make a bot that accepts images and applies filters (grayscale, blur, resize)
- [ ] Group Management Bot - Develop a bot for Telegram groups that can welcome new members, kick spammers, set rules, and moderate content.
- [ ] AI Chatbot Integration - Create a bot that integrates with an AI API (like OpenAI) to have natural conversations, answer questions, and provide intelligent responses.

## General Assignment
- [ ] URL downloader. accepts link for youtube, x (twitter), instagram, asks for quality and send the media file

## Notes
- Start with python-telegram-bot (PTB): It is the most well-documented library.
- Focus on Async: Ensure they learn async def and await syntax, as all modern libraries use this.
- Read the Docs: Telegram's API documentation is excellent; encouraged to read it alongside the library docs.
